# cdfy.us Funnel Build

Modern static JVZoo-style affiliate funnel build for the CDFY / Creation DFY offer.

Included pages:
- `index.html` — main landing / VSL page
- `checkout.html` — checkout handoff page
- `upsell.html` — OTO page
- `downsell.html` — lighter fallback offer
- `offer-wall.html` — thank-you / offer wall page

Notes:
- uses local placeholder SVG visuals for easy brand swaps later
- all CTA paths are site-first static handoffs for review
- replace button links with real JVZoo / checkout URLs later
